# 🚀 NEW TECH – Plataforma de Cursos Online

## 📌 Sobre o Projeto

O **New Tech** é uma plataforma web de ensino online desenvolvida para oferecer cursos digitais de forma simples, moderna e eficiente.

O sistema permite que usuários se cadastrem, realizem login, recuperem senha e, futuramente, adquiram cursos através de integração com WhatsApp.

---

## 🎯 Objetivo

* Centralizar a venda de cursos online
* Oferecer uma experiência moderna e responsiva
* Facilitar o acesso ao conteúdo educacional
* Permitir gerenciamento de cursos e usuários

## 🚀 Como Executar o Projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/SEU-USUARIO/newtech-cursos.git
```

### 2. Acessar a pasta

```bash
cd newtech-cursos
```

### 3. Instalar dependências

```bash
npm install
```

### 4. Rodar o projeto

```bash
npm start
```
