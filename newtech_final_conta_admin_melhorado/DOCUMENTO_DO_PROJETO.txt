PROJETO NEWTECH CURSOS

1. Objetivo
Desenvolver uma plataforma web para divulgação, inscrição e gerenciamento de cursos da NewTech.

2. Público-alvo
Alunos interessados em capacitação técnica e administradores responsáveis pela gestão dos cursos.

3. Funcionalidades principais
- Home institucional com informações da plataforma.
- Catálogo de cursos com pesquisa.
- Página de detalhes do curso.
- Login de aluno e administrador.
- Área do aluno exibindo apenas cursos inscritos.
- Inscrição realizada diretamente na plataforma.
- Área administrativa protegida.
- Cadastro, edição e exclusão de cursos.

4. Tecnologias
- React
- Vite
- React Router DOM
- CSS
- LocalStorage

5. Padrão visual
- Azul principal: #2563EB
- Azul secundário: #1E40AF
- Azul claro: #DBEAFE
- Cinza escuro: #1F2937
- Cinza médio: #6B7280
- Branco: #FFFFFF
- Laranja de destaque: #F97316

6. Fluxo do aluno
O aluno acessa a plataforma, visualiza os cursos, abre os detalhes, realiza a inscrição e é direcionado para o WhatsApp. Após a inscrição, o curso aparece na área do aluno.

7. Fluxo do administrador
O administrador realiza login com credenciais próprias, acessa a área ADM e gerencia os cursos exibidos na plataforma.

8. Melhorias futuras
- Banco de dados real.
- API backend.
- Pagamento integrado via Pix/cartão.
- Certificados digitais.
- Upload de aulas e materiais.

9. Atualização final
Foi adicionada pesquisa de cursos na Home, no catálogo e no painel administrativo. O processo via WhatsApp foi removido.
